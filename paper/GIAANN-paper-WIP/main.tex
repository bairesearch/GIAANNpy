\begin{abstract}
GIAANN Proto is an experimental implementation of a General Intelligence Algorithm Artificial Neural Network (GIA ANN) for language modeling.
The system constructs an excitatory database network of feature neurons organised into concept columns, incrementally updates that network during training, and performs autoregressive inference by predicting subsequent features conditioned on a seeded prompt.
This paper summarises the core ideas and provides an implementation-oriented description intended to accompany the released code.
\end{abstract}

\section{Summary}
Neural language models are commonly trained with gradient-based backpropagation and fixed-context attention mechanisms.
GIAANN explores an alternative design: an online, dynamically growing network of features and connections intended to support continuous learning and effectively unbounded context via a persistent database network.

This paper is a \emph{companion} to the codebase \texttt{github.com/bairesearch/GIAANNpy}.

\section{Background and Goals}
The project targets properties motivated by biological plausibility and continual learning, including online learning, continuous updates without catastrophic forgetting, a bias toward concept-level generalisation, and robustness to hallucination.

\section{Conclusion}
GIAANN Proto provides an experimental platform for exploring a persistent database-network approach to language modeling.
Future work should prioritise standardised evaluation metrics, scaling studies, and comparisons to strong modern baselines. Likewise, to move beyond self-supervised learning, reinforcement learning should be tested within the GIAANN architecture.
